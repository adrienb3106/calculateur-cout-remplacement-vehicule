# Comparateur de coût véhicule

Application React pour comparer le coût total de possession d'un véhicule actuel vs un nouveau véhicule (thermique ou électrique), en tenant compte du financement.

## Fonctionnalités

- Saisie des consommations et du prix de l'énergie pour chaque véhicule
- Paramétrage du financement : prix, négociation, apport, aides, reprise, durée et taux du prêt
- Résultats détaillés : tableau de coûts, graphe en barres empilées (carburant / entretien / prêt)
- Coût mensuel lissé dans le temps, avec comparaison avant et après la fin du prêt
- Données persistées dans le navigateur (localStorage)

## Lancer le projet

```bash
npm install
npm run dev
```
